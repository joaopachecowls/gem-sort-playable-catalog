Catálogo de Versões

O mesmo puzzle de revelar a imagem, em 8 estilos de beads diferentes. Cada versão é um playable completo (arquivo único, <5 MB) pronto para teste. Clique em uma versão para jogar.

Versões — formato dos beads
Sobre este catálogo. As 8 versões foram geradas a partir do mesmo build base (imagem revelada: cãozinho 🐶, logo GEM SORT, bolsa de veludo, CTA placeholder das lojas) — a única diferença entre elas é o conjunto de sprites dos beads. A versão de manutenção continua em dist/playable.html e é editada pela ferramenta Shift+H; este catálogo é regenerado com python tools/build_catalog.py.
